# Development

## Prerequisites

* PostgreSQL
* Ruby 2.4.1+

## Setup

### Install dependencies

```
bundle install
```

### Setup database

```
rake db:setup
```

### Run server

```
rails s
```

seperately run this in another tab for realtime reloading:

```
./bin/webpack-dev-server
```

The server is now running at http://localhost:3000
